Paintings from the Met Museum Collection

API: The Metropolitan Museum of Art Collection API

A collection of 470,000 artworks.

Holds all information and images pertaining to said artwork; all uploaded information can be used freely.

Options/Parameters

The two images you see above are two paintings in this massive collection. Since they had various works, I had to use a query string to get access to their paintings exclusively, and had to append a keyword to search for particular paintings. The queries used below will lead to these two paintings being associated with dog and cat respectively. To access each painting, I had to input the object id in this format: https://collectionapi.metmuseum.org/public/collection/v1/objects/[objectID]

Portrait of a Women

Query string with keyword: https://collectionapi.metmuseum.org/public/collection/v1/search?medium=Paintings&q=dog

Object ID: https://collectionapi.metmuseum.org/public/collection/v1/objects/437146

Cat Seen from Behind

Query string with keyword: https://collectionapi.metmuseum.org/public/collection/v1/search?medium=Paintings&q=cat

Object ID: https://collectionapi.metmuseum.org/public/collection/v1/objects/54747

Applications

  1. Can be used to randomly fetch a new artwork of the day to learn about
  2. Useful dataset for AI image generators to train on various artworks.
